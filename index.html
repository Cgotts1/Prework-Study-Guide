HTML

  • The head element contains information about the webpage.
  • The body element represesnts the visible content shown to the user.
  • A margin indicates how much space we want around the outside of an element.
  • A padding indicated how much space we want around the content inside an element

CSS

  • accent-color :Specifies an accent color for user-interface controls
  • align-content :Specifies the alignment between the lines inside a flexible container when the items do not use all available space
  • border-width : Sets the width of the four borders
  • color : sets the color of the text
  • margin :Sets all the margin properties in one declaration

Git

  • git status: checks what branch we are currently on
  • git checkout -b "branch-name": creates a new branch and switches to it
  • git init: creates a new local repostiory
  • git commit -m "Commit message": Commit changes to head (but not yet to the remote repository)
  • git branch -d "branchname": Delete the feature branch
  • git oush origin :"branch name" : Push the branch to your remote repository, so others can use it
  • git pull : Fetch and merge changes on the remote server to your working repository

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • An array is a special variable, which can hold more than one value.
  • A JavaScript Boolean represents one of two values: true or false.
  • JavaScript functions are defined with the function keyword.
  • Loops are handy, if you want to run the same code over and over again, each time with a different value.
  • JavaScript comments used to explain JavaScript code, make it more readable, prevent execution when testing alternative code.
